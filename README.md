# TodoList
🎮 基于Vue 3实现的我的世界风格待办事项应用

## 技术栈
- Vue 3（组合式API）
- Vuedraggable（拖拽排序）
- LocalStorage（数据持久化）
- CSS3（我的世界主题样式）

## 快速运行
1. 下载仓库所有文件；
2. 执行`npm install`安装依赖；
3. 执行`npm run dev`启动项目，访问本地地址即可使用。

## 核心功能
- 增删改查：添加/编辑/删除待办事项；
- 状态切换：标记完成/未完成，一键全选/取消；
- 拖拽排序：自由调整待办列表顺序；
- 数据缓存：本地存储，刷新页面不丢失数据。
